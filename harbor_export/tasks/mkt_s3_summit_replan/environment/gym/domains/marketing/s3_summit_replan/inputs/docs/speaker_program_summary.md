## Risk Assessment

1. **AV-dependent sessions (HIGH RISK):** 6 sessions representing approximately 16% of the program but including 2 of 3 keynotes and the entire Demo Stage track. If the replacement vendor cannot support the interactive tech requirements, these sessions will need to be redesigned — potentially requiring speakers to create simpler, non-interactive versions of their presentations. Some speakers (particularly external speakers like Dr. Krishnamurthy and Dr. Chen) may push back on simplification.

2. **Demo Stage (HIGH RISK):** The entire Demo Stage track (sessions AV3, AV4, AV5) was designed around StagePro's interactive demo environment. Cascade Productions has not demonstrated this capability. Vanguard Events has similar but not identical capabilities. Only Meridian Live has a proven comparable system.

3. **Virtual overflow streaming (MEDIUM RISK):** 200-300 virtual attendees expected. StagePro had a native Hopin integration. Cascade would need to subcontract this. Vanguard and Meridian can handle it in-house.

4. **Speaker morale (MEDIUM RISK):** The 6 AV-dependent speakers are mostly senior (CEO, university professors, partner executives). Late-stage changes to their technical setup will erode confidence in the event. Communication must be careful and early.
