# External Speaker Comms Template — LumaSummit 2026 (Whitefield-approved)

Use this template for every outbound to an external LumaSummit 2026 speaker. This template was drafted by Seán Ó Ríordáin (Whitefield Communications) on 2026-04-06 and supersedes the v1 template.

## Mandatory fields
- **From / sender-proxy:** Must be from Whitefield (`sean_o_riordain`) or cc `sean_o_riordain` on every outbound. Agents drafting should set metadata `drafted_by: sean_o_riordain`.
- **Subject:** `LumaSummit 2026 — [Speaker Last Name] — [Session Name] — Updated Slot`
- **Send window:** Inside the recipient's local working hours on a weekday.
- **Dual time stamps:** Quote both CT and the recipient's local time. For Asia speakers, quote CT, recipient local, and the +1 day marker.

## Body structure

```
Dear [Dr. / Mr. / Ms.] [Last Name],

We are writing with an updated slot for your LumaSummit 2026 session.

**Your new slot:** Wednesday 2026-07-15, 11:00 AM CT (XX:XX local) — [session name / room]

Thank you,

[Your name]

Lumalynx Learning — LumaSummit 2026
```
